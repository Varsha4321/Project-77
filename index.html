Project on console

Visual Studio is a software that allows creators and others build websites and code what ever they desire. You can set it up by downloading and installing it, createing a new file, and begin coding!

The console screen is a pop-up screen that allows others to "print" things on. It can also be used for debugging. You can open it by right clicking and clicking inspect. To print something on the console here are the steps. Go to your html code, open "script tag", write "console.log()" and pass whatever you want in the curved brackets.
